If running on WINDOWS:
- download the GAFindPhraseGivenTarget folder
- execute the GAFindPhraseGivenTarget.exe


ENTER TARGET PHRASE: insert the phrase you want the genetic algorithm to look for. (a-z characters + "space")
(NO special characters or accents)

ENTER POPULATION NUMBER: insert the population number (normal values are in the interval [10, 1000])

ENTER MUTATION PERCENTAGE: insert the mutation percentage (normal values are in the interval [0.01, 0.10]
